The goal is to provide clear, concise user-facing documentation from the
prompt.

- https://developers.google.com/tech-writing/resources

Rules:
- Keep related information together. If documentation exists on this topic,

- Ensure the information is accurate.
- If you need any clarification on common knowledge, use the web search tool.
- If you need any clarification from me, just ask.
